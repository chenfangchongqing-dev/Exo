U2 · Grammaire · La mise en relief

Transformez comme dans l’exemple. Mettez en relief le mot souligné de deux manières différentes.

Exemple : Cette mission m’a été très utile. → Ce qui m’a été très utile, c’est cette mission. / Cette mission, c’est ce qui m’a été très utile.

a. J’ai apprécié les échanges dans cette formation.
✅ Ce que j’ai apprécié dans cette formation, ce sont les échanges. / Les échanges, c’est ce que j’ai apprécié dans cette formation.
b. J’ai besoin d’aider les autres.
✅ Ce dont j’ai besoin, c’est aider les autres. / Aider les autres, c’est ce dont j’ai besoin.
c. Ce métier a changé ma vie.
✅ Ce qui a changé ma vie, c’est ce métier. / Ce métier, c’est ce qui a changé ma vie.
d. J’ai envie de travailler avec les enfants.
✅ Ce dont j’ai envie, c’est travailler avec les enfants. / Travailler avec les enfants, c’est ce dont j’ai envie.
e. Je voudrais m’engager.
✅ Ce que je voudrais, c’est m’engager. / M’engager, c’est ce que je voudrais.
